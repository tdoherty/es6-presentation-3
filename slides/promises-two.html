Promise Methods

  • Promise.resolve(value) - returns a promise that is resolved with the provided value
  • Promise.reject(reason) - returns a promise that is rejected with the provided reason
  • Promise.all(iterable) - returns a promise that resolves when all the iterable's promises have resolved
  • Promise.race(iterable) - returns a promise that resolves/rejects when one of the iterable's promises values has resolved/rejected
